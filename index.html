    Webエンジニアへ

    Webエンジニア「いっしー」のポートフォリオサイトです。

    私のことをもっと深く知ってほしいと思い、これまでに手がけた制作物、身につけたスキルをまとめています。

    どうぞごゆっくりサイト内を散策してください。

    開発スキル

    開発言語

    HTML / CSS / SCSS

    基本的なデザインであるフルスクリーンレイアウトや2カラム・3カラムなどのマルチカラムレイアウト、カード・タイル型レイアウトなど、WEBアプリケーションやWEBサイトの種類に合わせたデザインは得意です。

    また、マルチデバイスに最適化した画面表示ができるコーディングのレスポンシブデザインが可能です。

    JavaScript / jQuery

    Webサイトに実装されている基本的な機能や動きをCSSと組み合わせたイベント処理の実装が可能です。

    また、Webアプリケーションでは画面遷移せずにデータを取得できる非同期通信のAjaxはもちろん、オープンソースやライブラリなどを使用し、データを視覚化するチャートやグラフなど描画の実装なども可能です。

    Ruby / Ruby on Rails

    基本的な登録機能・参照機能・変更機能・削除機能であるCRUD機能をはじめ、一括処理機能やWebAPIを利用した機能など、目的に合わせた機能の実装が可能です。

    また、MVCアーキテクチャの設計思想を理解し、クラスメソッドとインスタンスメソッドを使い分け、DRY原則を意識した可読性の高い効率の良いコーディングを意識した実装も可能です。

    開発環境

    Personal Computer

    現在は3代目の自作パソコンを使用しています。

    元々は音楽制作ソフトに合わせたスペックのため、開発にはオーバースペックな「OS=>Windows10、CPU=>Intel Core i9-9900K、RAM=>32GB、SSD=>1TB」構成となっており、プロセッサは3.60GHzです。

    Storage

    ストレージは主に「Dropbox」を使用しています。

    クラウドストレージを用いることでパソコン内部にデータは残さずにすみ、ノートパソコンやデスクトップパソコンなど、状況に応じて開発がしやすいようにデータを管理しています。

    Editer

    エディターは主に「Visual Studio Code」を使用しています。

    あまり強いこだわりはないため、「AWS Cloud9(Cloud IDE)」や「Adobe Dreamweaver」など開発環境に合わせて使用しています。

    Source Code Management

    ソースコード管理は主に「GitHub」を使用しています。

    GitHubの主要な機能でリポジトリごとにバージョン管理したり、グループ機能で協働開発しています。

    その他

    Contents Management System

    ブログ運営は主に「WordPress」を使用しています。

    2018年からガジェット好きをターゲットにしたオウンドメディアを運用しており、有り難いことに今もなおPV数は少しずつ伸びて月間50,000PVを突破しております。

    Web Server

    ブログ運営やポートフォリオサイトのサーバーは主に「Xserver」を使用しています。

    Webサイトによってはサブドメインやサブディレクトリで運用するなど使い分けておりますが、コンテンツによってSEOに有利な構成になるよう構築しています。

    Access Analysis

    アクセス解析は主に「Googleアナリティクス/Googoleサーチコンソール/Rank Tracker」を使用しています。

    狙ったキーワードに対して意図した記事が上位表示されるようにデータ分析を行いながらコンテンツSEO対策をしています。

    プロフィール

    中小企業である人材会社で営業とキャリアアドバイザーからキャリアをスタートし、東証一部上場企業の総合商社である大手グループ企業で人事業務全般に従事。

    その後、HR Techのベンチャー企業やホスティングサービス業界でトップシェアを誇る企業にて人事組織の立ち上げに参画。人事機能の基盤作りを中心に人事戦略を立案し、採用戦略・教育企画・評価制度・労働環境の整備など多岐にわたり実施。

    様々な経験を経てモノづくりに携わりたい思いを実現すべく技術者として生きる道を決意し、人事マネージャーからWebエンジニアへジョブチェンジ。現在はWeb開発を中心に携わらせていただいています。
