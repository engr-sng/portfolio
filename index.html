    開発スキル

    死に物狂いで勉強したすえに身につけた開発スキルをグラフィカルにまとめてみました。

    広く浅くではありますが、フロントエンドからバックエンドまで幅広くスキルアップに取り組んでいます。

    インフラの構築回数は少ないですが、Webアプリケーションの本番環境はAWSを用いて構築しており、DRを考慮したAWSのインフラ構成やVPCを用いたネットワークの構築をしております。

    また、EC2のCPU高負荷時やステータス異常などの障害時にはCloudWatchでアラート設定、S3ログのバケットへはログが保存されるようにするなど、一通り実践しております。

    プロジェクト管理ツール

    概要

    DMM WEBCAMPの3ヵ月目のポートフォリオ制作カリキュラムで制作したプロジェクト管理ツールです。

    GitHub

    WebSite

    プロジェクト

    個人の目標管理を一箇所にまとめる記録アプリです。

    BacklogやRedmine、Wrikeなど組織でタスク管理やプロジェクト管理をするツールはたくさんありますが、個人で利用するにはコストも高くオーバースペックなことが多いです。

    近年では個人で仕事をする人が増加している背景もあるため、個人向けの目標管理ツールを制作しました。

    特徴的な機能

    • プロジェクト管理機能
    • メンバーアサイン機能
    • プロジェクト権限機能
    • ガントチャート機能
    • タスク一括登録、編集、削除機能
    • ステータスセレクト自動非同期通信機能

    使用言語など

    HTML,SCSS,Javascript,Jquery,Ruby,Ruby on Rails,AWS,Nginx,MySQL,Git,GitHubなど

    ECサイト

    概要

    DMM WEBCAMPの2ヵ月目のチーム開発カリキュラムで制作したECサイトです。

    GitHub

    プロジェクト

    プロジェクトメンバーは4名ですが、そのなかでもリーダーを担当させていただきました。

    PLとしての役割は以下となります。

    • プロジェクト管理
    • 基本設計、詳細設計を基準に仕様の方針調整
    • メンバーへのタスク割り振り
    • メンバーが解決できないエラーのデバック

    担当内容

    • 全体：開発用テストデータの準備
    • 全体：gem webpackの設定/管理
    • 管理者側：ログイン/ログアウト
    • 管理者側：会員編集機能
    • 管理者側：会員退会機能
    • 管理者側：注文履歴一覧/詳細/変更機能
    • 管理者側：ジャンル追加/編集機能
    • 顧客側：TOP/Aboutページ
    • 全体：全体のレイアウト調整
    • 全体：Localの日本語化対応
    • 全体：総合テスト/バグ修正

    使用言語など

    HTML,SCSS,Bootstrap,Javascript,Jquery,Ruby,Ruby on Rails,Git,GitHubなど
